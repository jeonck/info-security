# 양자 보안 (PQC, QKD 및 격자 기반 암호)

## I. 양자 컴퓨팅 위협과 양자 보안의 개요

**위협의 본질**: 쇼어 알고리즘(Shor's Algorithm)에 의해 소인수분해 및 이산대수 기반의 현대 비대칭 암호 체계가 실시간 해독될 위험 존재

**대응 방향**:
1. **양자 키 분배**(QKD): 양자 역학적 특성을 이용한 물리적 키 전달 보안
2. **양자 내성 암호**(PQC): 양자 컴퓨터로도 풀기 어려운 복잡한 수학적 난제를 활용한 알고리즘 보안

---

## II. 양자 보안의 핵심 기술 체계

### 가. QKD와 PQC의 비교 구조도

| 비교 항목 | 양자 키 분배 (QKD) | 양자 내성 암호 (PQC) |
|----------|-------------------|-------------------|
| **보안 원리** | 양자 역학의 물리적 특성 (중첩, 복제 불가능) | 수학적 난제 (격자, 다변수, 코드 등) |
| **구현 방식** | 전용 하드웨어 및 광케이블 필요 | 소프트웨어 업그레이드 (기존 인프라 활용) |
| **전송 거리** | 거리 제한 존재 (신호 감쇠) | 거리 제한 없음 (네트워크 전 구간) |
| **확장성** | 하드웨어 비용으로 인해 확장성 낮음 | 기존 암호 알고리즘 대체로 확장성 높음 |
| **주요 목적** | 안전한 키 교환 (Key Exchange) | 인증, 서명, 암호화 전 영역 |

### 나. PQC의 핵심: 격자 기반 암호 (Lattice-based Cryptography)

**개념**: `n`차원 격자 구조에서 "**최단 벡터 문제**(SVP)"나 "**최근접 벡터 문제**(CVP)"의 수학적 난해성을 이용한 암호 방식

**주요 알고리즘**: NIST 표준으로 선정된 Kyber(공용키 암호/KEM), Dilithium(디지털 서명) 등

**특징**:
- 양자 컴퓨터뿐만 아니라 고성능 슈퍼컴퓨터로도 해독이 어려움
- 연산 속도가 빠르고 키 사이즈가 상대적으로 효율적임
- 완전 동형 암호(FHE)의 기반 기술로도 활용됨

## III. 양자 보안 시대를 위한 단계별 대응 방안

| 단계 | 추진 과제 | 상세 내용 |
|:---:|----------|----------|
| 1. **하이브리드 적용** | 암호 민첩성(Crypto Agility) 확보 | 기존 암호와 PQC를 결합하여 보안성과 호환성 동시 확보 |
| 2. **인프라 고도화** | 국가 / 기업 핵심망 QKD 도입 | 금융, 국방 등 초고신뢰가 필요한 구간에 물리적 망 구축 |
| 3. **표준화 및 전환** | NIST 표준 알고리즘 채택 | 표준화된 PQC 알고리즘(Kyber 등)으로 시스템 점진적 교체 |
