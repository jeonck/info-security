# 공격 기법 & 침투 테스트 (Offensive Security)

윤리적 해킹과 공격자 관점의 보안을 다룹니다.

## 하위 분류

- **침투 테스트**: 방법론(PTES, OWASP), 도구(Metasploit, Burp Suite), 보고서 작성
- **취약점 연구**: 퍼징(Fuzzing), 리버스 엔지니어링, 익스플로잇 개발
- **레드팀 & CTF**: 레드팀/블루팀/퍼플팀 운영, CTF 문제 유형 및 풀이
